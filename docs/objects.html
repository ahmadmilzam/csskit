Object Intro

According to OOCSS Wiki, a CSS “object” is a repeating visual pattern, that can be abstracted into an independent snippet of HTML, CSS, and possibly JavaScript. That object can then be reused throughout a site.

Object Block

The Block Object simply stacks an image on top of some text-like content.

Block Default

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Necessitatibus provident facilis quod, voluptates ullam animi voluptatem enim quam ad aliquid totam vel dignissimos architecto laudantium voluptatum iure, modi. Voluptates, dolorem!

The Markup:

<div class="block">
  <div class="block__img">
    <img src="http://www.fillmurray.com/g/400/300" alt="fillmurray">
  </div>
  <div class="block__text">
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Necessitatibus provident facilis quod, voluptates ullam animi voluptatem enim quam ad aliquid totam vel dignissimos architecto laudantium voluptatum iure, modi. Voluptates, dolorem!</p>
  </div>
</div>
