Ratio object

Create ratio-bound content blocks, to keep media (e.g. images, videos, iframe) in their correct aspect ratios. http://alistapart.com/article/creating-intrinsic-ratios-for-video

Default 1:1


<div class="o-ratio">
  <iframe class="o-ratio__content" width="420" height="315" src="https://www.youtube.com/embed/aiBt44rrslw" frameborder="0" allowfullscreen></iframe>
</div>
